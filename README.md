# 🚀 TaskFlow Pro

A modern and premium productivity-focused To-Do Web App built using HTML, CSS, JavaScript, and Bootstrap 5.

TaskFlow Pro is designed to provide a beautiful and smooth task management experience with modern UI effects, responsive design, and useful productivity features.

---

## ✨ Features

* 🌙 Dark / Light Mode
* 💾 Theme Persistence using LocalStorage
* ✅ Add, Edit, Complete & Delete Tasks
* 📅 Due Dates
* 🏷️ Task Categories
* 🚨 Priority Labels
* 🔍 Search Tasks
* 🎯 Filter Tasks (All / Completed / Pending)
* 📊 Live Productivity Stats
* 📈 Dynamic Progress Bar
* 🎉 Confetti Celebration on Completion
* 🖱️ Drag & Drop Task Sorting
* 📱 Fully Responsive Design
* ✨ Smooth Animations & Hover Effects
* 🧊 Modern Glassmorphism UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Shreyansh063/taskflow-pro.git
```

### 2. Open Project Folder

```bash
cd taskflow-pro
```

### 3. Run the Project

Simply open:

```bash
index.html
```

in your browser.

You can also use VS Code Live Server for a better development experience.
